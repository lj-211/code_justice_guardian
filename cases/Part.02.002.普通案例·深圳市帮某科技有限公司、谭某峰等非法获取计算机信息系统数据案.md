# 深圳市帮某科技有限公司、谭某峰等非法获取计算机信息系统数据案

## 二、基本案情

### 公司及项目背景
被告单位深圳市帮某科技有限公司（法定代表人谭某峰）于2015年5月底成立"淘车大师"项目，研发同名手机APP，面向注册用户有偿提供车辆维修和保养记录查询服务。

### 获取数据的模式
被告人谭某峰、徐某（股东）、陈某通（员工）等人发现可通过4S店的DMS系统（经销商管理系统）获取维修保养数据。随后，公司安排员工联系各地不同汽车品牌4S店员工，以合作付酬为条件，让4S店员工私下提供DMS系统的账号和密码，并配合在其工作电脑上安装外挂软件，以便远程自动获取数据。

### 项目组织
2016年2月，徐某提出成立"215"项目组，专门负责通过外挂软件获取各地4S店DMS系统中的车辆维修和保养数据。陈某通负责研发部工作并与4S店员工联系安装外挂软件。

### 数据获取规模
- 通过**32个汽车品牌**的**68个账号**
- 累计查询 **1,800,799次** 车辆数据
- 成功查询 **1,556,590次**
- 成功查询的数据被保存在公司数据库服务器中，通过"淘车大师"APP对外成功查询 **8,571次**

### 违法所得
约 **12万元**。

### 4S店员工参与
被告人金某刚等4S店员工私自提供账户、密码，配合安装外挂软件，各自违法所得 **5,000元至55,500元** 不等。

## 四、裁判理由

### 行为定性
被告单位及被告人违反国家规定，未经授权或允许，侵入他人计算机信息系统，获取系统中存储的数据，情节特别严重，构成**非法获取计算机信息系统数据罪**。

### 4S店员工的角色
私自提供账户、密码并在内部工作电脑安装外挂软件，为获取数据提供帮助，情节严重，与被告单位及其主管人员、直接责任人员构成**共同犯罪**，系**从犯**。

### 关于"侵入"的认定（针对辩护意见）
- 刑法第285条第2款规定的"**侵入**"，**不限于**采用技术手段破坏系统防护进入系统。
- **包括**：未取得授权擅自进入、超出授权范围进入、以及**未经授权使用他人合法账号密码进入**。
- 本案中，被告单位虽未以技术手段突破安保措施，但**未经授权使用他人合法账号密码**登入系统后采用技术手段获取数据，符合"侵入""非法获取"的特征，具有社会危害性，应认定为犯罪。

## 五、裁判要旨

&gt; **串通公司员工，未经授权使用员工账号和密码，侵入公司计算机信息系统，通过植入外挂软件获取计算机信息系统中存储的数据，属于非法获取计算机信息系统数据；情节严重的，应当依照刑法第二百八十五条第二款的规定，以非法获取计算机信息系统数据罪定罪处罚。**

## 七、学习要点与启示

| 要点 | 内容 |
|------|------|
| **"侵入"的广义解释** | "侵入"不要求必须突破技术防护；未经授权使用他人合法账号密码进入系统，同样构成"侵入"。 |
| **内外勾结的共犯认定** | 内部员工提供账号密码并配合安装外挂软件，与外部公司构成共同犯罪，员工一般认定为从犯。 |
| **获取数据的合法性边界** | 即使数据本身不属于国家秘密或高度敏感信息，只要未经授权获取，且情节严重，即可入罪。 |
| **单位犯罪责任** | 公司决策、组织、受益的，构成单位犯罪，对单位判处罚金，对直接负责的主管人员和其他直接责任人员判处刑罚。 |
| **量刑因素** | 违法所得约12万元，查询次数超155万次，认定为"情节特别严重"；4S店员工所得较低，认定为"情节严重"并从犯，获较轻刑罚。 |

## 总结

本案明确了"**未经授权使用他人账号密码侵入计算机信息系统**"同样属于刑法意义上的"侵入"，即使未采用黑客技术突破防护，只要串通内部员工获取权限并植入外挂软件批量采集数据，即可构成非法获取计算机信息系统数据罪。该裁判规则对于打击内外勾结获取商业数据的行为具有重要指导意义。
