## 三、争议焦点与裁判理由

### 3.1 核心争议

**本案构成非法控制计算机信息系统罪，还是破坏计算机信息系统罪？**

### 3.2 两罪区分要点

| 对比维度 | 非法控制计算机信息系统罪 | 破坏计算机信息系统罪 |
|----------|------------------------|----------------------|
| **本质特征** | 未经授权或超越授权**控制**计算机执行特定操作 | **破坏**系统功能或数据，造成系统不能正常运行 |
| **行为方式** | 控制计算机实施特定操作 | 删除、修改、增加、干扰系统功能/数据/应用程序 |
| **结果要件** | 控制即可，不要求系统不能运行 | 需造成系统不能正常运行或影响正常运行 |
| **主观故意** | 意图控制计算机实施特定操作 | 意图破坏计算机系统 |
| **核心区别** | **控制** vs **破坏** | |

### 3.3 二审改判理由

#### （一）客观方面：属于"部分控制"而非"破坏"

1. **木马程序的功能定位**：
   - 操控计算机系统中的QQ程序
   - 向QQ用户发布广告引流
   - **仅控制计算机系统的涉QQ程序的功能**
   - 系**部分控制**计算机系统

2. **未造成破坏后果**：
   - 该行为**未造成计算机系统不能正常运行的严重后果**
   - 不符合破坏计算机信息系统罪的客观要件

#### （二）主观方面：控制意图而非破坏意图

- 各行为人**意在操控计算机系统的QQ程序并发布广告引流进而牟利**
- **而非意图破坏计算机系统**
- 不具有破坏计算机信息系统的犯罪故意

#### （三）主客观相一致原则

> 各行为人不具有破坏计算机信息系统的犯罪故意，其行为客观上也未造成计算机系统不能正常运行，不符合破坏计算机信息系统罪的犯罪构成。根据**主客观相一致原则**，各行为人的行为均构成**非法控制计算机信息系统罪**。

### 3.4 "情节特别严重"的认定

| 依据 | 内容 |
|------|------|
| **司法解释** | 《最高人民法院、最高人民检察院关于办理危害计算机信息系统安全刑事案件应用法律若干问题的解释》（法释〔2011〕19号）第1条 |
| **认定标准** | 违法所得已远高于人民币**25000元** |
| **量刑档次** | **情节特别严重**：三年以上七年以下有期徒刑 |

┌─────────────────────────────────────────────────────────────┐
│  非法控制计算机信息系统罪 vs 破坏计算机信息系统罪 区分规则   │
│                                                              │
│  非法控制计算机信息系统罪：                                  │

│                                                              │
│  破坏计算机信息系统罪：                                      │
│    → 核心在于"破坏"：造成计算机系统不能正常运行              │
│    → 或影响计算机系统正常运行                                │
│                                                              │
│  本案适用规则：                                              │
│    投放木马 → 部分控制系统（仅QQ程序）→ 实施非授权操作       │

## 六、案例启示

### 6.1 对司法实践的启示

1. **罪名区分的实务标准**：
   - 不能仅因使用了木马程序就认定为破坏计算机信息系统罪
   - 关键在于判断行为是"控制"还是"破坏"
   - 是否造成系统不能正常运行是重要区分标准

2. **"部分控制"的认定**：
   - 即使只控制了计算机系统中的某个应用程序（如QQ程序），也属于对计算机信息系统的控制
   - "部分控制"不影响非法控制计算机信息系统罪的成立

3. **二审改判的意义**：
   - 体现了罪刑法定原则和主客观相一致原则的贯彻
   - 罪名变更后，主犯刑期相应下调（何某骏减1年、蒋某阳减1年）
   - 说明罪名认定直接影响量刑

### 6.2 对辩护工作的启示

| 辩护要点 | 具体内容 |
|----------|----------|
| **罪名之辩** | 重点论证行为属于"控制"而非"破坏"，系统仍能正常运行 |
| **主观故意之辩** | 证明被告人意图是控制实施特定操作，而非破坏系统 |
| **情节之辩** | 审查违法所得计算是否准确，是否达到"情节特别严重"标准 |
| **作用地位之辩** | 区分主从犯，唐某、魏某、鲍某鑫、梁某胜可能认定为从犯 |
