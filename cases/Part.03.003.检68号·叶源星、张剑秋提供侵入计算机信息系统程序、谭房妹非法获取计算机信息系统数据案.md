# 叶源星、张剑秋提供侵入计算机信息系统程序、谭房妹非法获取计算机信息系统数据案

## 一、基本信息

| 项目 | 内容 |
|------|------|
| 案例编号 | 检例第68号 |
| 发布批次 | 最高人民检察院第十八批指导性案例（2019年4月） |
| 主题 | 严厉打击网络犯罪，共同防控网络风险 |
| 审理法院 | 浙江省杭州市余杭区人民法院 |
| 罪名 | 叶源星、张剑秋：提供侵入计算机信息系统程序罪；谭房妹：非法获取计算机信息系统数据罪 |

## 二、涉案人员

| 被告人 | 身份 | 角色分工 |
|--------|------|----------|
| 叶源星 | 超市网络维护员（1977年生） | 编写“小黄伞”撞库软件和打码软件，提供验证码识别服务 |
| 张剑秋 | 小学教师（1972年生） | 组织码工人工输入验证码（打码），与叶源星配合 |
| 谭房妹 | 农民（1993年生） | 使用“小黄伞”软件撞库，非法获取并出售用户信息 |

## 三、核心概念解析

| 术语 | 含义 |
|------|------|
| 撞库 | 黑客收集已泄露的用户信息，利用用户相同的注册习惯（如相同用户名和密码），批量尝试登录其他网站，非法获取可登录用户信息 |
| 打码 | 利用人工大量输入验证码的行为，用于绕过计算机系统的验证码安全保护 |
| 验证码 | 区分用户是计算机还是人的公共全自动程序，是常见的安全防护措施 |

## 四、案件事实

### （一）犯罪行为链条

```
叶源星编写"小黄伞"撞库软件（免费供他人使用）
    ↓
叶源星编写打码软件，通过网络有偿提供验证码识别服务
    ↓
张剑秋组织码工完成人工输入验证码任务，叶源星向其支付费用
    ↓
谭房妹下载使用"小黄伞"撞库软件，向叶源星购买打码服务
    ↓
谭房妹非法获取某电商平台用户信息2.2万余组，出售牟利
```

### （二）“小黄伞”软件的功能特征

- **用途单一**：仅针对某电商平台账号进行撞库和接入打码平台，无合法用途  
- **避开安全防护**：编入自动拨号功能，批量登录几组账号后自动切换新IP地址，防止被平台封锁  
- **绕过验证码保护**：自动抓取验证码图片发送到打码平台，由人工识别  
- **非法获取数据**：对登录成功的账号，自动抓取昵称、注册时间、账号等级等信息  

### （三）违法所得

| 人员 | 违法所得 |
|------|----------|
| 叶源星、张剑秋 | 人民币4万余元（出售验证码服务所得） |
| 谭房妹 | 人民币25万余元（出售用户信息所得） |

## 五、指控与证明犯罪

### （一）审查起诉阶段的难点与补侦

- **移送起诉时间**：2016年10月10日  
- **移送罪名**：非法获取计算机信息系统数据罪  

#### 辩护人提出的无罪意见：

- **叶源星辩护人**：批量验证已泄露信息不构成非法获取计算机信息系统数据罪  
- **张剑秋辩护人**：不清楚打码目的，与叶源星无共同犯罪故意  

#### 检察机关两次退回补充侦查（2016年11月25日、2017年2月7日）：

| 补侦内容 | 目的 |
|----------|------|
| 完善“小黄伞”软件的编制过程、运作原理、功能证据 | 明确是否具有避开/突破安全保护措施、非法获取数据的功能 |
| 对张剑秋电脑补充勘验，调取QQ聊天记录 | 确定张剑秋主观明知及二人犯意联络 |
| 提取叶源星电脑MAC地址，分析软件源代码 | 查明被非法登录账号与“小黄伞”软件的关联性 |
| 对谭房妹电脑和U盘补充勘验 | 确定用户信息是否系使用“小黄伞”软件获取 |

#### 关键补强证据：

- **技术特征认定**：证实“小黄伞”属于“专门用于侵入计算机信息系统的程序”  
- **主观明知**：QQ聊天记录提及“扫平台”、“改一下平台程序”、“那些人都是出码的”  
- **软件归属**：使用撞库软件的终端MAC地址与叶源星电脑MAC地址、软件源代码中的MAC地址一致  
- **数据归属**：谭房妹使用“小黄伞”获取的文件包含注册时间、账号等级等特有信息，与其他渠道获取的文件不同；登录时间与文件生成时间一一对应  

#### 罪名变更：

- 公安机关原以**非法获取计算机信息系统数据罪**移送  
- 检察机关审查后认为：
  - **叶源星、张剑秋 → 提供侵入计算机信息系统程序罪**
  - **谭房妹 → 非法获取计算机信息系统数据罪**

### （二）庭审辩论焦点

| 辩护意见 | 公诉人答辩 |
|----------|------------|
| 未提供省级以上有资质机构的检验结论，不足以认定“小黄伞”为专门侵入程序 | 在案电子数据、勘验笔录、技术人员证言、被告人供述等相互印证，足以认定 |
| 张剑秋与叶源星无共同犯罪故意 | QQ聊天记录反映二人提及非法获取用户信息内容，张剑秋主观明知，行为相互配合 |
| 违法所得应扣除支付给码工的钱款 | 违法所得以出售验证码服务金额认定，给码工的支出属于犯罪成本，不应扣除 |

### （三）判决结果

| 被告人 | 罪名 | 刑罚 |
|--------|------|------|
| 叶源星 | 提供侵入计算机信息系统程序罪（共同犯罪） | 有期徒刑三年，缓刑，并处罚金 |
| 张剑秋 | 提供侵入计算机信息系统程序罪（共同犯罪） | 有期徒刑三年，缓刑，并处罚金 |
| 谭房妹 | 非法获取计算机信息系统数据罪 | 有期徒刑三年，缓刑，并处罚金 |

> **特点**：三人均自愿认罪，退出全部违法所得，未上诉。

## 六、指导意义（核心要点）

### （一）“专门用于侵入计算机信息系统的程序”的认定

#### 审查认定的证据要求：

| 证据类型 | 具体内容 |
|----------|----------|
| 电子数据 | 从原始存储介质中收集、提取 |
| 勘验笔录 | 对涉案程序、被侵入系统及电子数据勘验、检查 |
| 书证材料 | 证实技术原理、制作目的、功能用途、运行效果 |
| 言词证据/视听资料 | 制作人、提供人、使用人对程序的阐述或功能展示 |
| 专业人员证言 | 证实被侵入系统安全保护措施的技术原理、功能及被侵入后果 |
| 侦查实验 | 对有运行条件的，应要求公安机关进行 |

#### 审查判断的三个维度：

1. **侵入目的**：结合安全保护措施，分析是否具有避开或突破功能  
2. **未经授权获取数据**：查明是否在未经授权/超越授权情况下获取数据  
3. **“专门”性**：分析是否属于专门用于侵入的程序  

#### 难以确定时的处理：

- 委托省级以上负责计算机信息系统安全保护管理工作的部门检验  
- 或由司法鉴定机构出具鉴定意见  
- 或由公安部指定的机构出具报告  

### （二）共同犯罪的认定

- 叶源星提供撞库软件 + 张剑秋组织打码 = 相互配合、相互补充的共同犯罪  
- 即使张剑秋未直接实施侵入行为，但其明知用于非法目的仍提供帮助，构成共犯  

### （三）违法所得的认定

- 犯罪成本（如支付给码工的费用）不应扣除  
- 共同犯罪中，各被告人对全部犯罪数额承担责任
