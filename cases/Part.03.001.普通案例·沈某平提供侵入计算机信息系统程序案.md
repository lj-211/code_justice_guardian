# 沈某平提供侵入计算机信息系统程序案

---

| 项目 | 内容 |
|:---|:---|
| **案号** | （2023）苏0981刑初153号 |
| **审理法院** | 江苏省东台市人民法院 |
| **裁判日期** | 2023年4月3日 |
| **罪名** | 提供侵入计算机信息系统程序罪 |
| **判决结果** | 有期徒刑三年，缓刑四年，并处罚金六千元；违法所得五万元没收 |

## 二、案件事实

### （一）行为时间
2021年1月至8月

### （二）涉案行为
1. **编写脚本程序**：针对"百某视频"APP的脚本软件系统，自行编写脚本程序
2. **技术功能**：
   - 突破"百某视频"APP的信息系统风控及反爬安全保护措施
   - 获取"百某视频"APP的用户信息
   - 实现对该APP用户的私信引流
3. **出售获利**：将程序出售给他人用于爬取用户信息并引流，**非法获利人民币5万元**

### （三）技术鉴定结论
- 可通过更换IP地址和帐号Cookie，突破信息系统风控、反爬安全保护措施
- 实现帐号的自动化批量登录
- 具有自动爬取特定信息的功能，实现**未经授权获取计算机信息系统数据**

---

## 三、争议焦点与裁判理由

### 核心争议
> 编写爬取用户信息的程序并向他人出售的行为，是否构成提供侵入计算机信息系统程序罪？

### 法院裁判要点（三个层次）

#### 1. "百某视频"APP属于"计算机信息系统"
- **法律依据**：《解释》第十一条第一款
- **认定理由**：
  - APP是安装在手机里的应用软件，属于访问终端
  - 需要连接网站后台进行逻辑操作
  - APP与其后台服务器等设备构成循环运行的计算机信息系统
  - 可进行数据储存、运算、传输，可通过移动通讯网络接入

#### 2. 案涉脚本程序属于"专门用于侵入计算机信息系统的程序"
- **法律依据**：《解释》第二条
- **认定标准**（需同时满足）：

| 特征 | 含义 | 本案体现 |
|:---|:---|:---|
| **侵入性** | 避开或突破安全保护措施，未经授权或超越授权 | 规避、突破风控和反爬措施，未经授权获取数据 |
| **专门性** | 专门用于违法犯罪目的，无法用于合法用途 | 专门针对被害公司平台编写，只可用于非法获取用户信息 |

#### 3. 出售行为属于"提供"行为
- 以获利为目的，编写并出售案涉脚本程序
- 符合刑法规定的"提供"行为要件

## 五、辩护要点与法院回应

### 辩护要点一：避开 IP 限制以及人机交互，不属于"未经授权"的范围

**辩护观点**：
- 更换 IP 地址和 Cookie 属于常规技术手段，未突破身份认证机制
- 人机交互验证（如验证码）仅为访问频率控制，不涉及权限边界
- 未破解加密协议或盗用账号密码，技术上仍属"公开访问"范畴

**法院认定**：
- 风控及反爬措施属于计算机信息系统的安全保护措施
- 通过技术手段规避上述措施，即构成"避开或者突破计算机信息系统安全保护措施"
- 批量获取用户信息未经平台授权，具有"未经授权获取计算机信息系统数据"的侵入性

**评析**：
> 法院将"反爬机制"纳入"安全保护措施"范畴，扩张了"侵入"的认定边界。实务中需注意：技术中立与刑事违法的界限在于**是否突破访问控制机制**，而非是否破解加密。

---

### 辩护要点二：APP 作为交互前端，不参与数据计算等核心逻辑，仅仅是 UI 层，不构成计算机系统的组成部分

**辩护观点**：
- APP 仅为用户界面（UI）呈现层，数据处理、存储均在后台服务器完成
- 单独的前端应用不具备"自动处理数据功能"，不符合《解释》第十一条"计算机信息系统"的定义
- 应严格区分"客户端"与"服务端"，避免将访问终端等同于系统本身

**法院认定**：
- APP 是安装在手机里的应用软件，属于**访问终端之一**
- APP 与其后台的服务器等设备构成了一个**循环运行的计算机信息系统**
- 该系统可以进行数据储存、运算、传输等工作，可通过移动通讯网络实现网络接入
- 整体属于《解释》中的"计算机信息系统"范畴

**评析**：

> **关于"计算机信息系统"的界定范围**：
> 
> 计算机信息系统的界定应着重于**实际处理信息的核心部分**，包括数据加解密、采集汇总、逻辑运算、过程流转及存储等关键环节。典型如环境监测系统中的数据采集模块、工程机械（如挖掘机）的控制锁定程序等，此类"重客户端"直接参与数据处理与控制，属于计算机信息系统的组成部分。
> 
> 然而，对于大量仅作为交互前端的 **APP 或 H5 页面**，其功能局限于用户界面（UI）呈现、输入转发及结果展示，**不参与信息计算、过程流转及核心数据存储**。将此类纯 UI 层扩大解释为"计算机信息系统"的组成部分，存在过度扩张刑法适用范围的风险。在分布式架构日益普遍的背景下，应严格区分"访问终端"与"处理核心"，避免将简单的界面交互层与实质的信息处理系统混为一谈。
> 
> **结论**：刑法第285条所保护的"计算机信息系统"应限定于具有**自动处理数据功能的核心系统**，而非无限延伸至所有与之连接的前端展示层。
